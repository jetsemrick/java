# Java workspace

This repository contains an **Apache HertzBeat** tree with an **Incident Command Center** demo layer.

- **Implementation**: [`hertzbeat/`](hertzbeat/) (cloned from [apache/hertzbeat](https://github.com/apache/hertzbeat)).
- **Demo guide**: [`hertzbeat/DEMO_INCIDENT_COMMAND.md`](hertzbeat/DEMO_INCIDENT_COMMAND.md).

**Build note:** upstream HertzBeat currently requires **JDK 25+** for Maven builds.
